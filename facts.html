About the Mandelbrot set

The Mandelbrot set is a fractal with infinite detail and complex patterns. It is a graph of z = z^2 + c. z starts at zero and c is any complex number. If a value of c never causes the function to "escape" to infinity, that point is colored black. If a point's value as c ever does escape, its color is assigned from an arbitrary list of colors depending on the number of iterations it takes for that point to escape.

Mandelbrot discovered this ground-breaking fractal when studying another type of fractal, Julia sets . These fractals can be defined by many different functions, but we can just use the same function that defines the Mandelbrot set, z = z^2 + c, where we are plotting all values of z for one constant c, instead of all values of c. These sets relate to the Mandelbrot set in a fascinating way. Since a Julia set can be either 1 or infinte pieces, if it is 1 piece, the orgin will be guaranteed to be black (not actually part of the Julia set but instead the "filled" Julia set, because the actual set is just the very edge of the filled area, but that's a lot to think about right now) and if it is infinitely many pieces, the orgin will be neither in the filled area or part of the set. However, when you assemble just the orgin points (or even just the complete graph) of each Julia set in a grid such that the position of the orgin point of the set corresponds to the set's value (c), you will quickly see a, you guessed it, Mandelbrot set emerge! How crazy.
